Getting Started

1. Clone or prepare your Git repository

Ensure your repo has a build script (e.g., pom.xml for Maven or build.gradle for Gradle).

2. Navigate to Code Repository

Add repositories and use Clone, Pull, or Fetch actions from the repository list.

3. Monitor in Pipelines

Track your build status in real time from the Pipelines section.

4. Stream Logs

Click Open Log on any run to stream live build output.

5. Review Artifacts

Successful builds store artifact paths in the Artifacts section.
